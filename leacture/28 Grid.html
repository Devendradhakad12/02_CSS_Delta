Note -> Grid is always used when we have to make row and column that is to make 2D layout, normally flex box is used. CSS Grid -> Setting a container's display to grid will make all childern grid items. Grid template -> They define the lines & track sizing. grid-template-rows:(100px 100px) ,( auto,auto); grid-template-columns:(100px 100px) , ( auto,auto); ------Repeat is used to divide all available space. grid-template-rows:reapeat(count,1fr) , repeat(2,1fr); grid-template-columns:reapeat(count,1fr) , 1fr 1fr; grid-gap -> They define the gaps between the lines. row-gap:10px; column-gap:20px; grid-gap:row column; Grid Columns -> Defines an item's starting & ending position inside the column. grid-column-start:line_number; grid-column-end:line_number; grid-column:start_col/end_col; grid-column:start_col/span number; Grid rows -> Defines an item's starting & ending position inside the rows. grid-rows-start:line_number; grid-rows-end:line_number; grid-rows:start_col/end_col; grid-rows:start_col/span number; -------Horizontal align justify-items:start,end,center;(container) justify-self:end,start;(item) -------vertical align align-items:start,end,center;(container) align-self:end,start;(item) -----both Horizontal & vertical align place-items:start,end,center;(container) place-self: align-self:end,start;(item) Display grid like inline element -> display:inline-grid; if we want to display grid container as a inline then we use inline grid
